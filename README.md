# ✨ Pixie Image Editor

Pixie is a lightweight, fast, and modern browser-based image editor built with **HTML, CSS, JavaScript, and PHP**. Designed for simplicity and performance, Pixie allows users to edit, crop, resize, and export images directly in the browser — with support for sizes from **16x16 up to 512x512**.

---

## 🚀 Features

- 🖼️ Upload and edit images instantly
- ✂️ Crop images with live preview
- 📏 Resize images (16px → 512px)
- 🎨 Background color & gradient support
- 🔲 Border & border-radius controls
- 🔤 Dynamic text/icon generation
- ⚡ Fast client-side rendering
- 📱 PWA-ready (installable on mobile & desktop)
- 💾 Export/download edited images

---

## 🧰 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Canvas API)
- **Backend:** PHP (no frameworks, no composer)
- **Optional:** Service Worker for PWA support
